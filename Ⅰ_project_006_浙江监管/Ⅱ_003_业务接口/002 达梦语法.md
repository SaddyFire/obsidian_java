### 01 日期函数
1. curdate()  -- 获取当前日期
```sql
//获取当前年月日
select curdate();
```
2. curtie() -- 获取当前时间
```sql
//获取当前精确的纳秒的时间
select curtime();
```
3. now() -- 获取当前日期时间
```sql
//获取当前年月日及其精确到纳秒的时间
select now();
```

5. add_months(date,num) ----在指定日期上增加指定的月数
6. dateadd(datepart,num,date) ----在指定日期的指定部分增加指定的数值
