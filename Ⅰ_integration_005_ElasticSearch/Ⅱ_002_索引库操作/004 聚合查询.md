## 01 Elastic控制台
#### 1.1 桶聚合
```json
# 统计每个分类下的文档数量

GET goods/_search
{
  "size": 0,
  "aggs": {
    "categoryAgg": {
      "terms": {
        "field": "categoryName",
        "size": 20
      }
    }
  }
}
```

#### 1.2 指标聚合

```json

# 统计品牌是苹果的所有文档的平均价格
GET goods/_search
{
  "query": {
    "term": {
      "brandName": {
        "value": "苹果"
      }
    }
  }, 
  "size": 0,
  "aggs": {
    "priceAgg": {
      "avg": {
        "field": "price"
      }
    }
  }
}

```
