安装时要注意: 
**忽略大小写**

```sql
-- 配置环境变量
DMSQL_HOMT
-- cmd进入
disql
-- 创建用户和密码  
create user INTERFACE1  identified by "interface123";  
-- 授权  
grant dba to DMSTAND;  
-- 创建表空间  (指定数据库存储位置)
create tablespace "DMSTAND" datafile 'D:\dmdbms\data\DMSTAND01.dbf' size 8192  CACHE = NORMAL;  
-- 规划表空间  
ALTER USER DMSTAND DEFAULT TABLESPACE "DMSTAND01";  
ALTER USER DMSTAND  DEFAULT INDEX TABLESPACE "DMSTAND";
```
