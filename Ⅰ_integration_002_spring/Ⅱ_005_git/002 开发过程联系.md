##### 1. 克隆
`git clone git@github.com:SaddyFire/interface.git`

##### 2. 修改代码前(Developer 角色)

1. 切换到开发分支
`git checkout develop`
2. 拉取开发分支
`git pull`
3. 建立自己的分支
`git checkout -b dev-wangdi `

##### 3. 修改代码后(Developer 角色)
1. 提交
`git commit -am '提交'`
2. 推送前先更新合并(有冲突先解决)
`git checkout develop`
`git pull`
`git merge develop`
3. 推送(同时建立远端分支)
`git push -u origin dev-wangdi`
